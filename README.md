# Running Locally
```bash
$ git clone https://github.com/tygerxqt/portfolio.git
$ cd portfolio
$ yarn
$ yarn run dev
```
Create a `.env` file, fill and replace the contents with:
```
DIRECTUS_URL=YOUR_DIRECTUS_URL
DIRECTUS_USERNAME=YOUR_DIRECTUS_USERNAME
DIRECTUS_PASSWORD=YOUR_DIRECTUS_PASSWORD
```
Make sure to change `YOUR_DIRECTUS_*` with the corrent details.
